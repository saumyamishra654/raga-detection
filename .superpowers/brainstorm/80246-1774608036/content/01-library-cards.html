Key elements

  • Card grid - 3 columns on desktop, 2 on tablet, 1 on mobile
  • Source badge - red "YT" or purple "File" in top-right of thumbnail area
  • Raga + tonic pills - shown once analysis is complete
  • Status indicators - green (complete), amber (processing with progress bar), red (error)
  • Ghost card - dashed "Upload or drop file" card always at the end of the grid
  • Drop zone - full-page overlay appears when dragging a file, then navigates to /upload with file attached
  • View toggle - Grid/List switcher in the action bar, persisted to user preferences
  • Filter pills - All, Processing, Complete, YouTube, Uploaded
  • Dark theme - deep navy/purple palette, consistent throughout app
