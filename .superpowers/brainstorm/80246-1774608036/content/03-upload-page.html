Key elements

  • Two tabs - "Upload File" and "YouTube URL", sharing the same form fields below
  • File tab - drag-and-drop zone + file browser. Pre-populated if user dragged a file onto the library
  • YouTube tab - URL input with live preview (thumbnail, title, channel, duration) once a valid URL is pasted
  • Title field - auto-populated from filename or YouTube title, editable
  • All optional fields default to "Auto-detect" - tonic, raga, instrument, vocalist gender
  • Collapsed advanced options - separator model, confidence thresholds, etc. (power user territory)
  • "Start Analysis" submits - redirects to library where the card appears in "Processing" state
  • Centered, narrow form - max-width 640px, clean and focused
